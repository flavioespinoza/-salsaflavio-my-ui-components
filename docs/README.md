# My UI Components

Tailwind-based UI components for React/Next.js.

## Installation

```bash
yarn add @salsaflavio/my-ui-components
```

## Usage

```tsx
import { Button, HomeIcon } from '@salsaflavio/my-ui-components'
```

## Tailwind Setup

```js
// tailwind.config.js
content: [
  './node_modules/@salsaflavio/my-ui-components/**/*.{js,ts,jsx,tsx}',
  './src/**/*.{js,ts,jsx,tsx}'
]
```
